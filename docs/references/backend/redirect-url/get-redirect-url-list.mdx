# `getRedirectUrlList()`

Retrieves a list of white-listed redirect URLs.

```tsx
function getRedirectUrlList(): () => Promise<PaginatedResourceResponse<RedirectUrl[]>>;
```

## `getRedirectUrlList()` example

```tsx
const response = await clerkClient.redirectUrls.getRedirectUrlList();

console.log(response);
/* In this example, you can see that data is an array of RedirectUrl objects, and is populated with one RedirectUrl.

      updatedAt: 1707151695693
    }
  ],
  totalCount: 1
}
*/
```
