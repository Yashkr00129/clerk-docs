# `disableUserMFA()`

Disable all of a user's MFA methods (e.g. OTP sent via SMS, TOTP on their authenticator app) at once.

```tsx
function disableUserMFA: (userId: string) => Promise<User>;
```

## `disableUserMFA()` parameters

| Name | Type | Description |
| --- | --- | --- |
| `userId` | `string` | The ID of the user to disable MFA for. |

## `disableUserMFA()` example

```tsx
const userId = 'user_2bxfCJOe0Ocd8DNe9hFN3EXvfOu';

const response = await clerkClient.users.disableUserMFA(userId);

console.log(response);
/* 
{ user_id: 'user_2bxfCJOe0Ocd8DNe9hFN3EXvfOu' }
*/
```
