# `deleteUser()`

Deletes a user given a valid ID.

```tsx
function deleteUser: (userId: string) => Promise<User>;
```

## `deleteUser()` parameters

| Name | Type | Description |
| --- | --- | --- |
| `userId` | `string` | The ID of the user to delete. |

## `deleteUser()` example

```tsx
const userId = 'user_2b8lKxzUgn5M1hsjaTW1mcjc0iB';

const response = await clerkClient.users.deleteUser(userId);

console.log(response);
/* 
_DeletedObject {
  object: 'user',
  id: 'user_2b8kQleSRNmcOSCdJ1Y8pSRr4mK',
  slug: null,
  deleted: true
}
*/
```
